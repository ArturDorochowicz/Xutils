1. Overview

Xutils plugin for PowerPro is a set of miscellaneous services.

2. List of services

  • LoadCd
  • EjectCd
  • DisableIdleTimers
  • EnableIdleTimers
  • SuDo
  • RunAs
  • ScrollUp
  • ScrollDown

3. Installation

The plugin must be placed in PowerPro's 'plugins' directory. After that its services can be used just like services from any other plugin. Read PowerPro's help for general information on using plugins.

4. Description of services

LoadCd( cdDriveLetter )
Close door of specified CD drive. Without argument closes door of the first CD drive in the system
  • cdDriveLetter - [optional] - letter of CD drive to use
EjectCd( cdDriveLetter )
Eject CD in specified CD drive. Without argument ejects first CD drive in the system
  • cdDriveLetter - [optional] - letter of CD drive to use
DisableIdleTimers( disableScreenSaver )
Disables idle timers. There are two idle timers: system idle timer and display idle timer. Calling this service prevents the display from turning off and the system from going asleep.
  • disableScreenSaver - [optional] - specify "1" to also disable screen saver
EnableIdleTimers( enableScreenSaver )
Enables idle timers. Please note that the system keeps track of disabling requests on a per thread basis and calling this service does not enable idle timers system wise.
  • enableScreenSaver - [optional] - specify "1" to also enable screen saver
SuDo( programPath, programArguments, workingDirectory )
Starts an application under built-in local administrator account.
  • programPath - [requiered] - path to executable to run
  • programArguments - [optional] - arguments for the executable
  • workingDirectory - [optional] - initial working directory for new process
RunAs( programPath, programArguments, userName, workingDirectory )
Starts an application using given credentials.
  • programPath - [requiered] - path to executable to run
  • programArguments - [optional] - arguments for the executable
  • userName - [optional] - name used in dialog prompting for credentials
  • workingDirectory - [optional] - initial working directory for new process
ScrollUp( scrollLinesCount )
Simulates scrolling up with mouse scroll wheel.
  • scrollLinesCount - [optional] - specifies speed of scrolling. If this argument is not given, the service reads this value from system settings.
ScrollDown( scrollLinesCount )
Simulates scrolling down with mouse scroll wheel.
  • scrollLinesCount - [optional] - specifies speed of scrolling. If this argument is not given, the service reads this value from system settings.
